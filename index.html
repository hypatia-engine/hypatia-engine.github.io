Hypatia is a game engine for single player 2D action adventure games, like The Legend Of Zelda: Oracle of Seasons and Oracle of Ages. Hypatia is aimed at non-programmers and programmers alike.

Sound interesting? Find out how to get started with Hypatia, without any programming knowledge.

See what Hypatia can do

Hypatia includes a simple demo game, which explores the basics of creating a game with the engine. You can see it in action below, or download it and have a look yourself.

Help Hypatia's development

Hypatia is developed by a group of volunteers, headed by Lillian Lemmer. You can help continue it's development by contributing to Lily's Patreon.

Want to help develop Hypatia? Just head over to our GitHub repository to get the code and start on your journey of contributing to an awesome open source project!
